快速开始
========

这是中文文档树下的快速开始页。旧英文快速开始仍保留在
:doc:`../getting_started`，用于兼容已有链接。

安装与构建
----------

.. code-block:: sh

   cmake -S . -B build
   cmake --build build
   cmake --install build --prefix install

顶层构建默认启用 ``BUILD_CLI``，因此会构建 ``featherdoc_cli``。如果只需要库，
可以显式传入 ``-DBUILD_CLI=OFF``。

最小 C++ 用法
-------------

.. code-block:: cpp

   #include <featherdoc.hpp>

   int main() {
       featherdoc::Document doc{"input.docx"};
       if (doc.open()) {
           return 1;
       }

       doc.paragraphs().add_run("Generated by FeatherDoc");
       return doc.save_as("output.docx") ? 1 : 0;
   }
