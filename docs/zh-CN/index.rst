FeatherDoc 中文文档
===================

Release v\ |version|

FeatherDoc 是一个 C++20 ``.docx`` 文档库，用于创建、读取、编辑、检查和
重写 Microsoft Word 文档。

这个入口保留主要阅读路径：快速开始、API 参考、PDF 工作流，以及当前
项目治理与长任务推进入口。旧的根目录长文档不再作为公开导航的一部分。
