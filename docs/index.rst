FeatherDoc
==========

Release v\ |version|

FeatherDoc is a C++20 library for creating, reading, editing, inspecting, and
rewriting Microsoft Word ``.docx`` files.

This documentation is organized as a product reference instead of a single long
README. Start with the API pages when you want to know which object to use,
which methods it exposes, and what parameters each workflow expects.
