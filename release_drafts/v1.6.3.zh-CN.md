# FeatherDoc v1.6.3

> 草稿状态：未发布  
> 适用分支：`dev`  
> 目标用途：后续 `v1.6.3` 正式 release 的中文正文底稿

## 核心变化

- 刷新了仓库对外展示面，README 与 Sphinx 首页现在直接展示 Word 渲染后的
  单功能 `merge_right()` / `merge_down()` fixed-grid 宽度闭环页，不再只靠
  聚合联系图表达表格闭环能力。
- 新增中文报价单模板的公开预览页，把 CJK 字体元数据、业务表格排版和真实
  Word 输出一起暴露到仓库首页和安装包的 `visual-validation/` 资源目录。
- 重建了 `README.zh-CN.md`，修复原来的乱码问题，恢复为可直接面向用户阅读的
  简体中文入口。
- 扩展了 `VISUAL_VALIDATION_QUICKSTART*.md`、
  `RELEASE_ARTIFACT_TEMPLATE*.md`、README 资源刷新脚本和安装清单，让
  这批新截图可以稳定进入安装包、发布说明和后续 release review 流程。
